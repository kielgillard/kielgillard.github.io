# Scriptural Rosary App

A web-based application for praying the scriptural rosary with biblical meditations.

## Features

- Four sets of mysteries: Joyful, Sorrowful, Glorious, and Luminous
- Daily mystery highlighting based on the day of the week
- Visual rosary bead progress tracking
- Scriptural quotes from NRSV for each mystery
- Full screen support for casting to TV
- Responsive design for desktop and mobile
- Smooth animated transitions

## Usage

1. Open `index.html` in a web browser
2. Select a mystery set from the home screen
3. Review the rosary structure overview
4. Click "Begin Prayer" to start
5. Navigate through prayers using Next/Back buttons
6. Double-click to enter/exit fullscreen mode

## Browser Support

Works in all modern browsers. For fullscreen casting, use Chrome or Firefox.

## Scripture Attribution

All biblical quotes are from the New Revised Standard Version (NRSV) and include proper book, chapter, and verse references.

## Resources

- **Scriptural Rosary PDF**: [A Scriptural Rosary by Thomas P. Rizzo](https://catholicconvert.com/wp-content/uploads/A-Scriptural-Rosary-English.pdf) - Contains scriptural references and reflections for each bead of the rosary.

## Reflections

The app includes reflections from St. John Paul II's apostolic letter "Rosarium Virginis Mariae" for each mystery, providing deeper theological insights into the mysteries of the rosary.
